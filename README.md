### 🧪 Lab 2 — Secure & Scalable VLAN Design (Router-on-a-Stick)

**ไฟล์:** `Group LAB/Lab 2.md`

**คำอธิบาย:**  
Lab 2 มุ่งเน้นการออกแบบ VLAN อย่างปลอดภัยและขยายได้ โดยใช้เทคนิค Router-on-a-Stick สำหรับ Inter-VLAN Routing

**หัวข้อหลักที่เรียนรู้:**
- Major Network: 192.168.10.0/24 แบ่ง Subnet ด้วย /26
  - VLAN 10 → 192.168.10.0/26
  - VLAN 20 → 192.168.10.64/26
  - VLAN 99 → 192.168.10.128/26
- Router-on-a-Stick ผ่าน 802.1Q Trunk (R1 G0/0)
- การตั้งค่า VLAN และ Trunk Port บน Switch
- PC-A (VLAN 10), PC-B (VLAN 20), PC-C (VLAN 99)
- Inter-VLAN Routing ด้วย Sub-interfaces บน Router

**ความรับผิดชอบ:** ณภัทร อรัญพูล — รับผิดชอบ Part 1

**เนื้อหา:**

| Network | รูปแบบ | คำอธิบาย |
|---------|--------|----------|
| Network 1 | Point-to-Point | เชื่อมต่อ PC สองเครื่องโดยตรง (192.168.0.x) |
| Network 2 | Star Topology (Switch) | ใช้ Switch เป็นศูนย์กลาง เชื่อม PC หลายเครื่อง |

**คำถามสรุป:**
1. เครือข่ายที่เล็กที่สุด = Point-to-Point
2. เครือข่ายที่ต้องใช้ Switch = LAN ที่มีหลายอุปกรณ์
3. เครือข่ายที่ต้องใช้ Router = เครือข่ายต่าง Subnet

### 📝 Assignment 4 — การวิเคราะห์โปรโตคอล (Protocol Analysis)

**ไฟล์:** `Personal/ASSIGNMENT 4_673380036-3.md`

**คำอธิบาย:**  
Assignment 4 ศึกษาการวิเคราะห์ Traffic ของโปรโตคอลต่าง ๆ ใน Cisco Packet Tracer Simulation Mode อย่างละเอียด

**เนื้อหาหลัก:**

**Part 1: สร้าง Traffic เพื่อเติมข้อมูลในตาราง**
- สร้างการจราจร ARP เพื่อเติมตาราง Address Resolution Protocol
- สร้างการจราจร HTTP (Web Traffic)
- สร้างการจราจร FTP (File Transfer)
- สร้างการจราจร DNS (Domain Name System)
- สร้างการจราจร Email

**Part 2: วิเคราะห์ TCP vs UDP**
- ศึกษา TCP Three-Way Handshake (SYN, SYN-ACK, ACK)
- วิเคราะห์ Port Numbers (SRC PORT, DEST PORT)
- ศึกษา Sequence Numbers และ ACK Numbers
- ทำความเข้าใจ TCP Flags
- แนวคิด Multiplexing — หลายแอปพลิเคชันส่งข้อมูลผ่านสื่อเดียวกัน

### 📝 Assignment 5 — Internet Edge + ISP Serial WAN + Microservices

**ไฟล์:** `Personal/Ass_5_673380036-3.md`

**คำอธิบาย:**  
Assignment 5 เน้นการ Transform Week03 Microservices เป็น Realistic Enterprise Deployment ที่ซับซ้อนขึ้น

**Topology:**
```
                     INTERNET
                        |
                    (DHCP)
                     R1 G0/0
                        |
                  [NAT OUTSIDE]
                        |
                 R1 S0/0/0 (100.10.10.1/30)
                        |
                 R2 S0/0/0 (100.10.10.2/30)
                        |
             -----------------------
             |                     |
     R1 G0/1 (LAN A)       R2 G0/1 (LAN B)
     192.168.10.1           192.168.20.1
             |                     |
         ServerA               ServerB
```

**หัวข้อหลักที่เรียนรู้:**
- R1 = Internet Edge Router (NAT + DHCP WAN)
- Serial WAN = ISP Domain (100.10.10.0/30)
- R2 = Branch Router
- Microservices deployed บน ServerA และ ServerB
- Cross-site Microservice Access ผ่าน Routed WAN + NAT
- Cisco CLI Configuration ทั้ง R1 และ R2
